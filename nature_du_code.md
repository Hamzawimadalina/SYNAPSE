# Nature du code
Le code présenté dans ce projet est une création originale de l'équipe. Il a été entièrement écrit par les membres du groupe, en respectant les notions vues en cours de NSI.
Nous avons utilisé Python et SQL, langages. Aucun code externe n'a été copié sans mention.
Concernant l'utilisation de l'IA, nous avons ponctuellement sollicité un assistant (ChatGPT) pour nous aider à structurer certaines parties du code, notamment pour la gestion des exercices.On a aussi utilisé DeepSeek afin de nous apprendre l'utilisation des bibliothèques `tkinter` pour l'interface graphique (incluse dans Python) et `matplotlib` pour les graphiques (installée via pip) et la fonction de hachage cryptographique sécurisée SHA-256 (Secure Hash Algorithm 256 bits)
Cependant, chaque proposition a été comprise, adaptée et intégrée par nos soins. 
L'IA a représenté moins de 10% du code final et n'a jamais été utilisée pour générer des parties entières sans notre compréhension. Nous restons transparents sur cette aide ponctuelle.
