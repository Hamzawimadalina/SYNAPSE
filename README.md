# SYNAPSE

SYNAPSE est une plateforme d'apprentissage adaptatif qui s'inspire de la plasticité cérébrale. Elle analyse les erreurs de l'élève, propose des fiches de révision ciblées et adapte le parcours pour une progression intelligente et durable.

## Prérequis

- Python 3.8 ou supérieur
- Les bibliothèques listées dans `requirements.txt`

## Installation

1. Clonez ou téléchargez ce dépôt.
2. Placez-vous dans le dossier du projet.
3. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
